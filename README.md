# py80Scan 局域网80端口扫描工具

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

基于 npcap 的端口扫描工具
它可以快速扫描指定网段内的 IP 地址, 检测是否开启80端口，如果开启则HTTP GET，并获取其网页标题以及根据MAC地址匹配设备厂商

相信可以帮助您轻松运维
